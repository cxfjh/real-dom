## 特性

- **轻量级** - 压缩后仅 10kb，零依赖
- **高性能** - 基于 Proxy 的深度响应式系统
- **组件化** - 声明式组件定义，支持模板、样式、逻辑分离
- **指令系统** - 内置丰富的指令（if、for、model、click 等）
- **路由管理** - 内置 SPA 路由，支持多容器
- **批量更新** - 自动批处理 DOM 更新，提升性能
- **精准依赖** - 智能依赖追踪和更新通知
- **简单易用** - 无需构建工具，直接引入即可使用

## 📦 安装
